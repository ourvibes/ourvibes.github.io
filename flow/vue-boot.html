vue 3.x 出来有一段时间了，一直没怎么上手。终于在昨天晚上试了一下，<script setup lang="ts">极大了解放了生产力，搭配 Vite，element-plus 使用起来的感觉超爽。于是我新建了一个仓库，专门新建了一个项目，用于对 Vue 项目搭建的实践，其中加入了在工作中常用的库，并封装了部分公用方法，力求在以后新搭建 Vue 项目时候，可快速进入开发阶段。

目前感觉，最新的 vue 偏向函数式编程，可以封装自定义 hook ，目前已经有了专门的 hooks 库 vueuse 可供使用。

  • element-plus
  • vueuse
  • vite
